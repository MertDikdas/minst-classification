# MNIST Digit Classification with PyTorch

This project is a deep learning image classification model built with PyTorch on Google Colab.  
The model classifies handwritten digit images from the MNIST dataset into numbers from 0 to 9.

## Project Goal

The goal of this project is to practice deep learning concepts such as:
- Data loading
- Model building with PyTorch
- Training and evaluation loops
- Accuracy and loss calculation
- Prediction visualization

## Technologies Used

- Python
- PyTorch
- Torchvision
- Google Colab
- Matplotlib

## Model Architecture

The model uses a Convolutional Neural Network architecture inspired by TinyVGG.

## Results

The model was trained and evaluated on the MNIST handwritten digit dataset.

## How to Run

1. Open the notebook in Google Colab
2. Run all cells
3. Check training and evaluation results
